# Рекомендации по внесению вклада

Спасибо за интерес к проекту! Мы приветствуем вклад в любой форме.

## Как внести вклад

### Отчеты об ошибках

Если вы нашли ошибку, пожалуйста, создайте issue с описанием:
- Подробное описание ошибки
- Шаги для воспроизведения
- Ожидаемое поведение
- Фактическое поведение
- Версию PHP и Laravel (если применимо)

### Предложение улучшений

Для новых функций или улучшений:
- Обоснуйте необходимость
- Предложите возможное решение
- Примеры использования

### Pull Requests

1. Fork репозиторий
2. Создайте ветку для функции (`git checkout -b feature/amazing-feature`)
3. Внесите изменения с понятными commit messages
4. Убедитесь, что все тесты проходят
5. Push ветку на GitHub
6. Откройте Pull Request

## Стандарты кода

- Используйте PSR-12 стандарт
- Добавляйте PHPDoc для всех публичных методов
- Пишите тесты для новых функций
- Поддерживайте существующие тесты в работоспособном состоянии

## Запуск тестов

```bash
composer install
vendor/bin/phpunit
```

## Лицензия

Все вклады лицензируются под MIT лицензией.
